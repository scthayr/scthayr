## About

I'm a front-end developer working mainly with Vue and React, focused on clean component architecture and interfaces that feel calm rather than cluttered. Figma bridges my design and code work, and I lean on AI tools to move faster without cutting corners.

Right now I'm deepening my front-end fundamentals and experimenting with AI-assisted workflows through personal projects.
